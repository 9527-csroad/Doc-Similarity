# 需求与思路备忘（书籍相似 / 热门）

## 当前已实现（v2）

1. **上传**：`pdf_url` 临时下载 → SHA256 哈希 + PDF 元数据（ISBN 等）→ 不落盘 PDF；`txt_url` 全文用于指纹与本地 `data/texts/{book_id}.txt`。
2. **指纹**：伪页切分；伪页数 ≥8 时取前 3 + 中 3 + 后 2 + 标题；不足 8 伪页则用全文。`merged`（拼接后一次 embedding）与 `pooled`（分段 embedding 加权平均）两套向量均入库（两套 FAISS）。
3. **查重**：`content_hash` / `isbn` / 向量（按请求 `fingerprint_mode` 选索引），命中则只增上传记录、不写新指纹。
4. **检索**：`books/search` 用已存指纹向量检索，阈值过滤。
5. **热门**：`books/hotlist` 按时间窗统计；`threshold<1` 时按指纹相似度合并分组。
