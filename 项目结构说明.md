---

**定位**：FastAPI 服务；对外 **仅 v2 书籍接口**。PDF 临时下载算哈希与元数据；正文来自 `txt_url`；**伪页指纹**（merged + pooled）写入 **两套 FAISS**；全文落盘 `data/texts/`。旧版 v1 文档上传 API 已移除；`tasks/pipeline.py` 等仍保留供历史/测试，主业务以 `book_pipeline` 为准。

---

## 根目录

| 路径 | 作用 |
|------|------|
| `app/` | 应用主代码 |
| `alembic/` | 数据库迁移 |
| `requirements/`、`requirements.txt` | 依赖 |
| `docker-compose.yml` | 依赖编排 |
| `alembic.ini` | Alembic 配置 |
| `.env` / `.env.example` | 环境变量 |
| `extract_dataset_texts.py`、`import_test_dataset.py` | 工具脚本 |
| `tests/` | 测试 |
| `data/` | 运行时：`docsim.db`、`texts/`、指纹 FAISS 等 |

---

## app/

| 路径 | 作用 |
|------|------|
| `main.py` | FastAPI 入口；仅挂载 **v2**；`/health` |
| `config.py` | DB、Embedding、`TEXTS_BASE_PATH`、`CHARS_PER_PSEUDO_PAGE`、双指纹 FAISS 路径等 |

### api/

| 路径 | 作用 |
|------|------|
| `v2/books.py` | 上传、相似搜索、热门 |

### schemas/

| 路径 | 作用 |
|------|------|
| `book.py` | v2 请求/响应（含 `fingerprint_mode`） |
| `document.py`、`search.py`、`config.py` | 旧 v1 模型文件仍存在，**未**从 `schemas/__init__.py 导出** |

### models/

| 路径 | 作用 |
|------|------|
| `book.py` | `Book`、`BookUpload`（含 `text_path`、`pseudo_page_count`） |
| `document.py` | 旧文档表 ORM（迁移仍存在） |

### processors/

| 路径 | 作用 |
|------|------|
| `pseudo_pages.py` | 伪页切分、指纹片段 |
| `fingerprint.py` | merged / pooled 向量计算 |
| `bge_embedding.py` 等 | Embedding |
| `pdf_processor.py` | 书籍流水线中仅用于 **临时 PDF 元数据** |

### services/vector/

| 路径 | 作用 |
|------|------|
| `faiss_store.py` | FAISS；可传自定义 index/meta 路径 |
| `__init__.py` | `get_vector_store()`（旧文档流）、`get_fingerprint_store(mode)`（书籍） |
| `milvus_store.py` | Milvus；**书籍指纹未走 Milvus** |

### tasks/

| 路径 | 作用 |
|------|------|
| `book_pipeline.py` | 书籍上传主流程：哈希、全文落盘、双指纹写入 |
| `pipeline.py` 等 | 旧文档处理链（非当前 HTTP 主路径） |

---

## data/（运行时）

- `texts/{book_id}.txt`：上传正文  
- `faiss/fingerprint_merged.bin` + `fingerprint_merged_meta.json`  
- `faiss/fingerprint_pooled.bin` + `fingerprint_pooled_meta.json`  
- `docsim.db` 或 PostgreSQL：元数据  

---

**重点**：HTTP 入口 `app/api/v2` → `book_pipeline` → `pseudo_pages` / `fingerprint` → `get_fingerprint_store`；配置见 `app/config.py` 与 `接口文档.md`。
